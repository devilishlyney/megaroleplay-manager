# MegaRolePlay Manager — Guide Utilisateur

Application de gestion de personnages et campagnes D&D.

## Compte
- **Inscription/Connexion** : Écran d'accueil → *Sign Up* ou *Login* avec e-mail/mot de passe.
- **Profil** : Modifiable via le profil. Avatar cliquable pour upload.

## Création de personnage (Wizard)

1. **Race & Classe** : Sélection via boutons. Affiche traits et caractéristiques.
2. **Statistiques (Point Buy)** : 27 points à répartir sur 6 caractéristiques (Force, Dextérité, Constitution, Intelligence, Sagesse, Charisme). Valeurs entre 8 et 15. Les 27 points doivent être dépensés.
3. **Apparence** : Éditeur visuel par couches (base, oreilles, yeux, bouche, cheveux, frange). Navigation via flèches ← →.
4. **Histoire** : Zone de texte libre pour le lore.
5. **Résumé** : Vérification puis clic sur *Créer*.

## Personnages
- Liste accessible via l'onglet *Personnages*.
- Clic sur un personnage → modification ou suppression.

## Campagnes
- **Créer** : Onglet *Campagnes* → *Créer*. Champs : nom, description, statut.
- **Détails** : Clic sur une campagne.
- **Ajouter un personnage** : Bouton *+ Ajouter*. Sélection depuis la liste (les déjà assignés sont grisés).
- **Retirer un personnage** : Bouton *Retirer* à côté du personnage.
- **Supprimer** : Bouton *Supprimer la campagne* (les personnages ne sont pas supprimés).

## Amis
- **Rechercher** : Barre de recherche par nom d'utilisateur ou nom d'affichage.
- **Demander** : Bouton *Ajouter* sur un résultat de recherche.
- **Demandes reçues** : Section *Demandes en attente* → *Accepter* ou *Refuser*.
- **Profil d'un ami** : Clic sur le nom dans la liste d'amis.
- **Retirer** : Bouton *Retirer* à côté de l'ami.

## Thème
- Bouton bascule clair/sombre dans l'en-tête. Sauvegardé automatiquement. Suit le thème système au premier lancement.
